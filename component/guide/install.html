# 安装

使用本组件库需要使用 Vue 2，可以安装 Vue CLI 4.0.X 脚手架

安装 Vue CLI 4.0.X，打开终端运行下列命令：

npm i ifeather-ui -g

或

yarn global add cifeather-ui
